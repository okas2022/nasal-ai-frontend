# 연세대학교 청각재활연구소 — Nasal AI Frontend

본 페이지는 **Hugging Face 백엔드(`okas2000/nasal-ai-backend`)**와 연동되어  
비강 내시경 이미지를 업로드하면 **용종/하비갑개 비후/협착/정상** 판독과  
지표(확신도, 비후 등급, 협착도, 발적/광택/엣지밀도 등)를 시각화하여 보여줍니다.

- 프론트: GitHub Pages (이 저장소)
- 백엔드: https://okas2000-nasal-ai-backend.hf.space/api/predict

## 사용법
1. 이미지를 업로드 (JPG/PNG)
2. **AI 분석 시작** 클릭
3. 결과 카드/표/레이더 차트로 판독 결과 확인
