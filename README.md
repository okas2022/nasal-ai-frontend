# 연세대학교 청각재활연구소 비강 AI 분석기 (Frontend)

이 프로젝트는 Hugging Face 백엔드(`nasal-ai-backend`)와 연동되어 작동합니다.

## 🚀 주요 기능
- 이미지 업로드 및 AI 분석 결과 시각화
- 비후 등급, 협착도, 홍조지수 등 주요 지표 표시
- 반응형 UI 및 Tailwind 스타일 적용

## 🧭 배포 방법
1. GitHub에 업로드
2. Settings → Pages → Branch: `main` → Folder: `/ (root)` 설정
3. 저장 후 `https://{사용자명}.github.io/nasal-ai-frontend/` 접속
